# Legacy batch scripts archive

このディレクトリは、現行バッチへ統合された旧バッチを削除せず退避するためのアーカイブです。

## 退避したファイルと置き換え先

- `run_base_pipeline.ps1`
  - 置き換え先: `../build_master_range.ps1`
- `run_scrape_then_build_by_date.ps1`
  - 置き換え先: `../run_scrape_build_raceinfo_range.ps1`

## 退避理由

- 旧バッチは現行バッチとスコープが重複しており、運用導線を新バッチへ集約するため。
- 互換性確認や参照のため、削除ではなくアーカイブとして保持する。

## 復活の判断基準

以下のように、旧運用を再現する必要が明確な場合にのみ復活を検討する。

- 旧バッチの実行順序・引数・ログ出力を含めた既存運用の再現が必要な場合
- 新バッチでは代替できない過去手順の検証が必要な場合
